Manual GIT

  • Se crea un archivo en el computador donde se va a almacenar la informacion
  • Se abre el archivo en la herramienta que vaya a a usar (Visual Code - Sublime Text)
  • Se crea la cantidad de archivo que vaya a guardar en el GITHUB
  • Una vez creado los archvios, empezaremos a crear la conexion con GITHUB
  • Se abre el terminal del Visual Code
  • Se inicia un nuevo repositorio con el comando (git init)
  • Se establece el estatu de los archivo con el comando (git status)
  • Se adiciona los archivo que desea el usuario con el comando (git add .)
  • Con el comando (git status) se observa la situacion de los archivos
  • Una vez hecho al paso anterior, establecemos la conecion con el GITHUB
  • Creamos el repositorio en la web de GITHUB
  • Una vez creado establecemos la conecion con el repositorio de la web
  • Para establecer la conecion se usa el comando (git remote add origin https://github.com/bacabrales/Ejemplo1.git)
  • Una vez establecida se usa el comando (git remote -v) para ver la coencion
  • Se usa el comando (git push -u origin master) para subir los archivos
  • Usamos el comando (git push) para terminar la verificacion de los archivos
